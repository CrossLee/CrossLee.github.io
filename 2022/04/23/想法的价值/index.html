想法的价值

善于思考的人脑子里动不动就会产生各种想法。善于记录的人时时刻刻都在将有价值的东西记录在本子上。善于收藏的人可能就是将各类碎片收藏到浏览器的收藏夹。我是一个三类都沾边的人，却不是一个将某一类贯彻到底的人。所以面对那么多的收藏和笔记，我想了想，为什么不分享出来让大家都能有所收获呢？躺在收藏夹里和笔记里的碎片永远没有价值，当其能为某些人产生启发就产生了价值，当其能为某些人产生收入就产生了价值，当然如果能为社会做出贡献那可是不小的社会价值。

当然只通过我一个人的分享那是远远不够的，毕竟个人的视野、认知、知识储备都是有上限的。希望能借助所有人的力量为所有人输出有价值的东西。所以借助 GitHub 这个平台，我抛砖引玉建立一个 OIS（Old Ideas Store）Repo，通过这个 Repo 分享「曾经赚过钱」的 Idea，它不一定能让你直接赚钱，毕竟都是老的 Idea，但是我希望能对读者有所启发。

Repo 地址：https://github.com/CrossLee/Old-ideas-store
